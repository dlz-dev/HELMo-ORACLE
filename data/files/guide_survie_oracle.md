# Guide de Survie des Nouveaux Aventuriers

Bienvenue, jeune voyageur. Ce guide contient les connaissances de base pour ne pas périr lors de votre première nuit dans les Terres d'Exil.

## Équipement de Départ Essentiel
Pour bien commencer, assurez-vous de posséder :
- **Une gourde d'eau** (se remplit aux fontaines sacrées).
- **Un silex** pour allumer un feu de camp protecteur.
- **Une boussole ancienne** qui pointe toujours vers le Puits de Vérité.

## Les Trois Règles d'Or
1. Ne jamais s'aventurer dans la forêt après le coucher du soleil sans une source de lumière.
2. Toujours saluer les marchands de la guilde avec la main gauche levée.
3. Si vous entendez un murmure appeler votre nom, ne vous retournez pas.
