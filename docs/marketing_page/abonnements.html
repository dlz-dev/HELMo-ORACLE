Qu'est-ce que le MCP ?

MCP (Model Context Protocol) est le protocole qui permet à Oracle d'agir, pas seulement de répondre. Sans MCP, un bot est une encyclopédie. Avec MCP, Oracle devient un agent — il peut créer, envoyer, modifier, et déclencher des actions dans vos outils métiers.

C'est une boucle ReAct (Reasoning + Acting) : Oracle analyse la demande, choisit l'outil approprié, exécute l'action, observe le résultat, et recommence jusqu'à avoir répondu au besoin. L'utilisateur voit une réponse fluide. En coulisses, Oracle a potentiellement créé un ticket, envoyé un email, et mis à jour une base de données.

Resources — La donnée

Sources d'information consultées en temps réel : règlements, catalogues, calendriers, bases de données. Le bot sait toujours quoi répondre car il accède à vos données à jour.

Tools — L'action

Fonctions exécutables : créer un ticket, envoyer un email, générer un document, interroger une API. C'est là qu'Oracle passe du "chat" à l'"agent".

Prompts — Le raisonnement

Modèles de comportement pré-configurés pour chaque contexte métier. Un prompt "étudiant Erasmus" rend le bot multilingue et pédagogique. Un prompt "juridique" le rend précis et sourcé.
