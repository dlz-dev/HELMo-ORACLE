Le chef d'orchestre de l'Oracle. LangGraph nous permet de structurer le raisonnement de l'IA sous forme de graphe d'états.

  • ◈ Mémoire conversationnelle persistante.
  • ◈ Routage intelligent selon la question posée.

Frontend & UI

Une interface épurée conçue avec Tailwind CSS, exploitant le Glassmorphism pour fondre la technologie dans l'univers de Dofus.
