Mystères Dévoilés

Les réponses aux questions les plus courantes sur HeLMO Oracle.

Général

Comment l'Oracle garantit-il l'exactitude des faits de lore ? ▼
C'est le cœur de l'architecture RAG. L'Oracle ne "génère" pas une réponse à partir de ses propres "souvenirs". Lorsqu'une question est posée, il interroge d'abord notre base de connaissances certifiée (stockée sur Supabase) pour trouver les documents les plus pertinents. Il utilise ensuite ces documents comme unique source de contexte pour synthétiser sa réponse.
Les données du lore sont-elles à jour ? ▼
Oui. Notre base de connaissances est continuellement mise à jour en se basant sur les dernières publications officielles, les patch notes du jeu et les articles du Dofus Wiki. Un processus de validation est en place pour assurer la cohérence des informations.
L'Oracle peut-il répondre à des questions sur le jeu Dofus (classes, builds, etc.) ? ▼
La mission principale de l'Oracle est de maîtriser le lore du Krosmoz. Bien qu'il puisse avoir connaissance de certains aspects du gameplay via les patch notes, il n'est pas optimisé pour donner des conseils de build ou de stratégie de classe.

Technique

Quels modèles de langage (LLMs) l'Oracle utilise-t-il ? ▼
Notre architecture LangGraph est agnostique du modèle. Par défaut, nous utilisons Groq (Llama 3) pour sa rapidité incroyable, mais l'interface permet d'utiliser n'importe quel LLM compatible (OpenAI GPT-4o, Anthropic Claude, Gemini, Mistral, etc.). Nous supportons même des modèles locaux via Ollama.
Qu'est-ce que LangGraph et pourquoi l'utiliser ? ▼
LangGraph est une librairie qui permet de construire des agents LLM complexes sous forme de graphes. Nous l'utilisons pour orchestrer le pipeline RAG : recevoir la question, la reformuler, chercher les documents, vérifier leur pertinence, et finalement générer une réponse sourcée. Cela rend le système plus robuste et observable.
La base de données vectorielle est-elle auto-hébergée ? ▼
Nous utilisons Supabase avec l'extension `pgvector`. C'est une solution cloud managée qui offre d'excellentes performances pour la recherche vectorielle tout en simplifiant la maintenance. Pour les plans "Légende", un déploiement sur une infrastructure privée (On-Premise) est possible.

Abonnement & Contribution

Qu'est-ce que la recherche "Hybride" proposée dans l'abonnement ? ▼
Pour une précision maximale, nous combinons deux méthodes de recherche sur Supabase : la recherche sémantique (via pgvector, pour comprendre le sens de la question) et la recherche Full-Text classique (BM25, pour les mots-clés exacts). Les résultats sont fusionnés via l'algorithme RRF (Reciprocal Rank Fusion) pour un classement optimal.
Puis-je utiliser l'Oracle pour mon propre projet/guilde ? ▼
Absolument ! Le plan "Légende" est spécifiquement conçu pour cela. Nous pouvons intégrer vos propres documents (archives de guilde, stratégies, etc.) dans une instance privée de l'Oracle et le déployer sur votre Discord ou site web. Contactez-nous pour discuter de votre projet.
Comment puis-je soumettre une correction ou une source manquante ? ▼
La communauté est essentielle. Le meilleur moyen est d'ouvrir une "Issue" sur notre dépôt GitHub en précisant la correction et, si possible, la source officielle. Notre équipe examinera votre contribution et mettra à jour la base de connaissances.
